1️⃣ A file dairy-backup-DATE.json downloads to your phone

📥 Step 2 — How to Restore from WhatsApp
On your iPhone / Android:
1️⃣ Open WhatsApp → tap your own name (Saved Messages)
2️⃣ Find the backup message → long press the file attachment
3️⃣ Tap "Share" or "Open in…" → tap "Save to Files"
4️⃣ Come back to Dairy Manager → Accounts → 💾 Backup
5️⃣ Tap ⬆️ Import below → select the saved .json file
6️⃣ Wait for import report → tap OK to reload ✅
